Greta Shop — agentic storefront

Quick start

Open the site locally by running one of these commands from the project root:

Python 3 built-in server:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000/greta-shop/ in your browser.

Files

- `index.html` — main page
- `src/css/style.css` — styling
- `src/js/app.js` — assistant and cart logic

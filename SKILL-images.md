# GretaShopImageUpdates

Use this skill when the request is about adding, replacing, or improving images on the page.

## Scope
Apply this skill to updates involving:
- hero images or featured visuals
- product photos
- section illustrations or decorative images
- image cards and visual banners
- adding imagery to improve the look of existing sections

## What to do
1. Identify which element should receive the image.
2. Add the image in a way that fits the current layout and visual tone.
3. Keep the page feeling soft, playful, and boutique-style.
4. Preserve accessibility by using meaningful alt text.
5. Keep the implementation lightweight and responsive.

## Preferred approach
- Prefer existing layout patterns and reusable card styles.
- Use images that feel cohesive with the warm, calm palette.
- Avoid clutter and keep the imagery balanced with the surrounding text.
- Ensure images scale well on mobile and desktop.

## File targets
- HTML changes: index.html
- Styling changes: src/css/style.css
- Optional asset files: add images under a new assets or images folder if needed

## Example requests
- Add an image to the hero section
- Add product photos to the catalog cards
- Add a visual banner to the assistant section
- Replace a placeholder with a more polished image
